# ReactTestSite

Aplicação de loja virtual em React com visual responsivo, consumo de API externa e cadastro local de produtos.

## Tecnologias

- React
- Vite
- Axios
- PrimeReact
- PrimeIcons
- PrimeFlex

## Funcionalidades

- consumo da Fake Store API para listar produtos
- exibição em cards estilizados com PrimeReact
- formulário para adicionar novos produtos localmente
- busca por nome e descrição
- filtro por categoria
- ordenação por destaque, nome e preço
- favoritos
- contador de carrinho
- atualização manual da lista de produtos
- layout responsivo

## Como executar

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```
